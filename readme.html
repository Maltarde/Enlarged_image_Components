Composant s'agranssiment d'image

Utilisation

  1. Copier le contenue du composant dans le code
  2. Il faut mettre "@click="fullSize(image)"" sur les images que l'on veut agrandir
  3. Il faut mettre la classe "cursor-full-size" sur les images que l'on veut agrandir
  4. Choisir le style

Style

Il y a trois configuration possible full, flottant et simple. Pour active l'un d'entre eux il suffit de mettre dans la classe "full", "float" "simple"

Mobile

On peut rajouter la classe "m-full" pour mettre la configuration en full pour le mobile. Et choisir à quelle taille d'écran le basculement se fait en changant le pixel du media queiry
